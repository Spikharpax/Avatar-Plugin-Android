Client mobile Android pour Avatar

Utilisez vos téléphones ou vos tablettes Android pour communiquer avec le Serveur Avatar depuis n'importe où dans votre maison.
Vous pouvez aussi l'utiliser comme un clone et passer une règle dans une pièce où se trouve déjà un client fixe Windows.

Quelques fonctionnalités:
Une superbe interface avec plusieurs thèmes de couleurs et très simple d'utilisation.
La possibilité d'avoir plusieurs serveurs et de changer par un simple clic.
Entièrement compatible avec tous les plugins, y compris les plugins comprenant des dialogues de questions/réponses.
Si vous avez un système de son Sonos, le son peut être redirigé vers les Players de vos pièces plutôt que sur l'enceinte du téléphone.
Un menu navigateur pour exécuter des actions directement sans les vocaliser. Ce menu, très simple, est entièrement configurable sans aucun développement.
